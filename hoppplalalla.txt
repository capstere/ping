Varför dialogens “Flikar som signeras” blir tomt

I din nuvarande kod byggs listan i Main.ps1 så här:

$wsTargets = @()

if ($doWsGransk) { $wsTargets += "Granskning (...)" }

Det här är inte flikarna som faktiskt signeras i worksheeten — det är bara “mode-rader”. När något i flödet gör att $wsTargets blir @() (eller innehåller tomma strängar), får du en tom dialog.

Samtidigt är den faktiska signeringslogiken i OpenXmlHelpers.ps1 → Invoke-WorksheetSignature_OpenXml dynamisk och kan:
	•	hoppa över flikar som saknas
	•	hoppa över Resample Data Summary om HasResample är false
	•	hoppa över Data Summary-flikar om de saknar data (DataSummaryGuard)
	•	osv.

✅ Rätt lösning är därför:
Bygg dialogens Targets från samma OpenXML-beslut som signeringen använder, fast i “read-only plan/dry-run” (utan att skriva).

+function Get-WorksheetSignaturePlan_OpenXml {
+    <#
+      Dry-run: beräknar vilka worksheet-flikar som skulle signeras för ett mode.
+      Bygger på samma targetlista/guards som Invoke-WorksheetSignature_OpenXml.
+      Returnerar: object med Planned (List[string]) och Skipped (List[string]).
+    #>
+    param(
+        [Parameter(Mandatory=$true)][string]$Path,
+        [Parameter(Mandatory=$true)][ValidateSet('Sammanstallning','Granskning')][string]$Mode,
+        [switch]$HasResample,
+        [Parameter(Mandatory=$true)][string]$ModulesRoot
+    )
+
+    Set-StrictMode -Version 2.0
+    Import-OpenXmlSdk -ModulesRoot $ModulesRoot | Out-Null
+
+    $res = [pscustomobject]@{
+        Mode    = $Mode
+        Planned = New-Object System.Collections.Generic.List[string]
+        Skipped = New-Object System.Collections.Generic.List[string]
+    }
+
+    $doc = $null
+    try {
+        $doc = [DocumentFormat.OpenXml.Packaging.SpreadsheetDocument]::Open($Path, $false) # read-only
+        $wbp = $doc.WorkbookPart
+        if (-not $wbp -or -not $wbp.Workbook) { throw 'WorkbookPart saknas.' }
+        $sheets = $wbp.Workbook.Sheets
+        if (-not $sheets) { throw 'Sheets saknas.' }
+
+        # OBS: Håll denna targetlista i sync med Invoke-WorksheetSignature_OpenXml.
+        $targets = @()
+        if ($Mode -eq 'Sammanstallning') {
+            $targets = @(
+                @{ Name='Test Summary';                DataSummaryGuard=$false },
+                @{ Name='Data Summary';                DataSummaryGuard=$true  },
+                @{ Name='Extra Data Summary';          DataSummaryGuard=$true  },
+                @{ Name='Resample Data Summary';       DataSummaryGuard=$true;  ResampleOnly=$true },
+                @{ Name='Resample Date Summary';       DataSummaryGuard=$true;  ResampleOnly=$true },
+                @{ Name='Seal Test Failure Count';     DataSummaryGuard=$false },
+                @{ Name='Statistical Process Control'; DataSummaryGuard=$false },
+                @{ Name='Vacuum Seal Data';            DataSummaryGuard=$false }
+            )
+        } else {
+            $targets = @(
+                @{ Name='Test Summary'; DataSummaryGuard=$false }
+            )
+        }
+
+        foreach ($t in $targets) {
+            $sheetName = $t.Name
+
+            if ($t.ContainsKey('ResampleOnly') -and $t.ResampleOnly -and (-not $HasResample)) {
+                [void]$res.Skipped.Add("$sheetName (ej resample)")
+                continue
+            }
+
+            $sheet = @(
+                Get-OpenXmlChildrenOfType -Parent $sheets -Type ([DocumentFormat.OpenXml.Spreadsheet.Sheet]) |
+                Where-Object { $_.Name -and $_.Name.Value -eq $sheetName }
+            )[0]
+            if (-not $sheet) {
+                [void]$res.Skipped.Add("$sheetName (saknas)")
+                continue
+            }
+
+            $wsp = $wbp.GetPartById($sheet.Id)
+            if (-not $wsp) {
+                [void]$res.Skipped.Add("$sheetName (saknar WorksheetPart)")
+                continue
+            }
+
+            if ($t.DataSummaryGuard) {
+                $mergeIndexes = $null
+                try { $mergeIndexes = Get-MergeIndexes_OpenXml -WorksheetPart $wsp } catch { $mergeIndexes = $null }
+                $mergeMap = if ($mergeIndexes -and $mergeIndexes.ContainsKey('Forward')) { $mergeIndexes.Forward } else { $null }
+                $hasData = Test-OpenXmlDataSummaryHasData -WorksheetPart $wsp -WorkbookPart $wbp -DataColLetter 'C' -MergeMap $mergeMap
+                if (-not $hasData) {
+                    [void]$res.Skipped.Add("$sheetName (ingen data)")
+                    continue
+                }
+            }
+
+            [void]$res.Planned.Add($sheetName)
+        }
+    } catch {
+        try { Gui-Log "⚠️ Kunde inte planera Worksheet-signering: $($_.Exception.Message)" 'Warn' } catch {}
+    } finally {
+        try { if ($doc) { $doc.Dispose() } } catch {}
+    }
+
+    return $res
+}
